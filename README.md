# TV LOGO PO DRŽAVAMA (120x120)

Ovaj repozitorij sadrži folderima organizirane TV logotipove po državama i kategorijama.

## Sadržaj

- Svaki folder predstavlja određenu državu ili kategoriju.
- U folderima se nalaze slike logotipa u formatu PNG.
- Dimenzije slika su 120x120 piksela.
